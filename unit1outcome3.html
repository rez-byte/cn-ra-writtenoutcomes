3.2 Describe ways to improve productivity and efficiency

One way in which I was able to improve my productivity was by using a free extension from the extension marketplace set up by Microsoft inside VS code. The extension I used was called Live Server and seems to be quite popular among the user of VS code and as I previously had mentioned this extension saved me a lot of time by allowing me to check the adjustments and changes I made in my code editor without having to go through a too roundabout or finicky manner, by simply displaying a live version of the site with updates as I went. This can be especially useful when using HTML and CSS as error message aren’t directly shown as they often are in python. So through this method I can see real time my errors and swiftly fix them. Another method I’ve begun quite regularly to improve efficiency is to memorise certain repeating lines of code and using its shortcuts so that I lessen my keyboard travel allowing me to to type faster and without much pause.

An example of this would be a single text print in html requires the following: first a less than & more than bracket sign with p written inside it and to end the same but with a forward slash then a p written inside. This can all be quickly skipped because of VS code's intuitive system that allows for you to simply type the letter p and then pressing tab button will complete all of the code that is at the type of the dropdown list that appears which start with the most used codes first. this applies to most parts of CSS aswell and considerably speeds up the entire process.
